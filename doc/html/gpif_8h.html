int gpif_init ( gpif_session *  session,
char *const  argv[] 
)

Create a gnuplot session.

A call to this function will fork/exec a gnuplot child process, set up pipes to it's STDIN/STDOUT accoringly and populate the supplied session with the necesary information.

Parameters
sessionSession to be initialized
argvFirst item: name of the gnuplot binary, will usually be 'gnuplot' Any successive items: gnuplot arguments The last item must always be NULL
Returns
EGPIFOK No errors occured
EGPIFINVAL An invalid argument has been passed
EGPIFERR Session could not be created
